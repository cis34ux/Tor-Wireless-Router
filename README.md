# Tor Wireless Router With RaspberryPi 3
Bash script to automate the creation of a Tor WiFi access point.

## Install
```
chmod +x install.sh
sudo ./install.sh
```

## Start
```
chmod +x start.sh
sudo ./start.sh
```

## Stop
```
chmod +x stop.sh
sudo ./stop.sh
```
